# RS-31_Evtushenko_Volodymyr_RTOS_LR6
Номер групи: РС-31

Автор: Євтушенко Володимир

ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32 БІТНІ МК Лабораторна робота 6 Широтно-імпульсна модуляція для управління силовими
елементами на основі апаратних таймерів

Обладнання: STM32F411, CubeIDE.

Мета: Ознайомитися з принципами роботи широтно-імпульсної модуляції (ШІМ) на
STM32F411, навчитися використовувати апаратні таймери для керування силовими елементами
та реалізувати програму з керуванням шириною імпульсу за допомогою кнопки

Завдання до лабораторної роботи:

Варіант 5.Запрограмувати одночасну генерацію ШІМ на PD12, PD13, PD14, PD15 з різними частотами (1
кГц, 2 кГц, 500 Гц, 5 кГц), кнопка PA0 керує duty cycle тільки PD12

Функціонал:
Програма ініціалізує мікроконтролер та запускає PWM-сигнали на таймерах TIM1, TIM2, TIM3, TIM4 (канал 1).

Таймери генерують ШІМ-сигнали.
На таймері TIM4 реалізовано керування скважністю через кнопку.
